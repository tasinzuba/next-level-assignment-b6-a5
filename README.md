# Movie Portal Backend

A RESTful API for a Movie and Series Rating & Streaming Portal built with Node.js, Express, Prisma, and PostgreSQL.

## Tech Stack
- Node.js + Express.js
- Prisma ORM
- PostgreSQL (Neon)
- JWT Authentication
- bcryptjs

## Setup
```bash
npm install
npx prisma generate
npm run dev
```
